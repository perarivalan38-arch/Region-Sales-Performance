# Region Sales Performance Dashboard using Microsoft Excel

## Project Overview

- This project demonstrates how Microsoft Excel can be used to create an interactive Sales Performance Dashboard for business analysis.

- The dashboard transforms raw sales data into meaningful KPIs and visual reports, helping stakeholders monitor sales performance and identify business opportunities.

---

## Dashboard Features

- Interactive Excel Dashboard
- Pivot Tables
- Pivot Charts
- Dynamic Slicers
- KPI Cards
- Regional Sales Analysis
- Product-wise Sales Analysis
- Sales Person Performance
- Profit Analysis

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization

---

## KPIs

- Total Sales
- Total Units Sold
- Total Profit
- Average Sales

---

## Business Insights

- Identified highest-performing sales regions.
- Compared product-wise sales performance.
- Evaluated sales representative performance.
- Monitored profit trends.
- Enabled interactive business reporting.
